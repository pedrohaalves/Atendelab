# AtendeLab

Sistema de Controle de Atendimentos Acadêmicos desenvolvido na disciplina de Design e Arquitetura de Software.
Este projeto tem como objetivo criar uma plataforma web acadêmica simples e eficiente, focada em registrar, acompanhar e consultar os atendimentos realizados por professores, monitores, laboratórios, setores de apoio e equipes acadêmicas.

## Tecnologias utilizadas

- PHP 8.x
- MySQL
- phpMyAdmin
- HTML
- CSS
- Bootstrap
- Git e GitHub

## Funcionalidades previstas

- Página pública
- Login
- Dashboard
- Cadastro de pessoas atendidas
- Cadastro de tipos de atendimento
- Registro de atendimentos
- Relatórios
